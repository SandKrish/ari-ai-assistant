# Ari – AI Assistant with WhatsApp Integration

## What it does
AI assistant that helps manage tasks, automate workflows, and respond via WhatsApp.

## Tech Stack
- Python
- React 18
- Vite
- Whisper-STT
- Ollama (LLM)
- Flask
- Google Tools(Sheets, Calender)
- Twilio API
