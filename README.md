# Ari – AI Assistant with WhatsApp Integration

## 🚀 What it does
Ari is an AI-powered assistant that helps manage tasks, automate workflows, and respond intelligently via WhatsApp.

## 🧠 Key Features
- WhatsApp-based interaction using Twilio
- AI responses powered by Ollama (LLM)
- Voice input support using Whisper (Speech-to-Text)
- Task automation and workflow handling
- Integration with Google Sheets and Calendar

## 🛠️ Tech Stack
- Python
- Flask
- React 18 + Vite
- Ollama (LLM)
- Whisper (Speech-to-Text)
- Twilio API (WhatsApp Integration)
- Google APIs (Sheets, Calendar)
